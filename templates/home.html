Project Overview

Aluminum Alloy Casting Quality Estimation is a machine learning-powered system designed to automate and improve the quality inspection process in aluminum alloy casting. It uses sensor and process parameters to classify each sample as Acceptable or Defective.

This solution not only enhances production efficiency and reduces manual inspection effort, but also ensures higher reliability using automated predictions and intelligent analytics.

Algorithms Used

  • 🔹 Support Vector Machine (SVM) - used for robust binary classification with RBF kernel.
  • 🔹 XGBoost - gradient boosting decision tree model for high accuracy and interpretability.
  • 🔹 Logistic Regression (optional) - for baseline comparison and probability outputs.

Dataset Details

The model is trained on a balanced dataset containing process parameters like Pouring Temperature, Mold Temperature, Injection Speed, and chemical compositions (Al, Si, Cu, etc.). The dataset is sourced from industrial simulations and quality-labeled.

Key Features

📊 Performance Metrics

Accuracy, Precision, Recall, F1 Score, AUC – all displayed clearly after training.

📈 Visual Analytics

Confusion Matrix, ROC Curve, Correlation Heatmap to understand data and model performance.

⚙️ Real-Time Prediction

Use the form to input new casting values and instantly estimate the output quality.

Workflow Summary

  1. 📥 Upload or load the dataset from `media/` folder.
  2. ⚙️ Train models: SVM, XGBoost, and optionally Logistic Regression.
  3. 📁 Save trained models and scalers to `media/models/`.
  4. 📊 Generate and store graphs to `media/graphs/` (ROC, Confusion Matrix, Correlation Heatmap).
  5. 📝 Use prediction form to submit new values and display output with confidence score.
